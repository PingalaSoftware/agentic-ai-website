# Images Directory

Place your images in this directory.

## Required Images:

- `hero-bg.jpg` - Background image for the hero section (recommended size: 1920x1080px)

## Optional Images:

- Conference images
- Speaker photos
- Sponsor logos

## Notes:

- Images referenced in the code will fallback to CSS gradients if not present
- For optimal performance, use optimized WebP format
- Recommended hero image: Dark, tech-themed background with conference or AI visualization
